Project Goal

This page outlines the core objectives for FactorZ at ShellHacks 2025. Our team is working to solve a real-world problem while leveraging incredible tools provided to us, thanks to the event sponsors: Google ADK and Google A2A.

Objective

 Using various AI agents, we aim to build a system that delivers faster, more accessible storm warnings and situational awareness. A user selects a location on an interactive map, and the system retrieves the longitude and latitude. That data is checked against a database of radar towers provided by the National Weather Service.

 Once the appropriate radar tower is identified, another agent requests the reflectivity map of the area. The radar image is then analyzed by an agent to assess storm patterns. If a dangerous weather event is detected, the system determines the severity and returns a set of outputs, including:

  • The city and state of the selected location
  • The assessed danger level
  • Immediate recommended actions (evacuation or other measures depending on severity)
  • The radar reflectivity map image as supporting proof
  • The latitude and longitude originally provided by the user

Goal

 The ultimate goal is to create a faster response system for weather hazards. Government-issued warnings often face delays, which puts people at risk. Additionally, underprivileged areas may not receive timely alerts or any alerts at all. By automating analysis and response, this system seeks to empower communities with immediate, actionable weather intelligence.

Milestones

  • Interactive map with selection-based analysis
  • Automated image analysis and reporting
  • Scalable ingestion and agent orchestration
